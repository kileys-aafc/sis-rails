CanSIS Development Website (sistest.agr.gc.ca)

This site is used to generate the CanSIS website. You can test the generated pages here

  • See config/routes.rb and app/views for how the different portions are generated.
    • Listings of all files in a given directory are found using urls such as this one.
    • Listings of all files generated by this application are found here.
  • Updating of DEV.SIS and SIS is done by copying any updated MySQL databases and Rails files:
    • Upload the current working directory branch of the Rails project using "pub"
    • Upload a MySQL database or table by running "mysql.dump" and following the instructions
  • Common Look and Feel is added using content from /app/views/layouts
